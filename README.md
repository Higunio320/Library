
# Library
Simple web app made to learn how to integrate database with backend and frontend.

# Tech stack
1. Backend
   * Java 17
   * Spring Boot 3.1.5
   * Maven
   * Hibernate
   * Postgresql
   * Lombok
   * Mapstruct
   * JJWT
    
2. Frontend
   * Angular 17
   * TypeScript
   * SCSS
   * Angular Material
   * Nginx

3. Others
   * Docker
